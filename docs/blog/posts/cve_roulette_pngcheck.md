## 关于 CVE Roulette

这个系列的创意很简单：随机选取一个历史 CVE 漏洞，进行深入分析和复现，编写漏洞 POC / EXP，并记录完整的分析过程。在分析目标漏洞的同时，也可能会对相关产品进行更全面的审计，期待发现新的问题。
